
# KPI 정의서


```
본 문서는 Olist E-Commerce 데이터 분석 프로젝트에서 사용되는 핵심 KPI와 파생 KPI의 정의와 계산 및 활용 기준을 명확히 하기 위해 작성되었습니다.
모든 분석 및 대시보드는 본 문서의 KPI 정의를 기준으로 수행합니다.
```


---

## 1. 핵심 KPI


```
핵심 KPI는 커머스 비즈니스의 전반적인 상태를 요약하고, 모든 분석 프레임워크에서 공통적으로 활용되는 핵심 지표입니다.

- 총 매출 (Gross Revenue)
- 주문 수 (Order Count)
- 구매자 수 (Active Buyers)
- 재구매율(Repeat Purchase Rate)
```


### 총 매출(Gross Revenue)

- **정의**
	- 결제가 완료된 주문을 기준으로 산출한 총 주문상품 금액(상품 금액 + 배송비)

- **비즈니스적 의미**
	- 커머스 비즈니스의 최종 성과 지표로, 매출의 성장·정체·감소 여부를 판단하기 위한 핵심 지표

- **계산 대상(Granularity)**
	- 주문

- **계산 기준**
	- 주문에 포함된 주문상품 금액의 합계

- **포함 조건**
	- 배송이 완료된 주문만 포함

- **시간 축 활용**
	- 연 / 월 / 분기 단위 매출 추이
	- YoY / MoM 성장률 분석
	- 구조적 변화 및 시즌성 탐지


### 주문 수 (Order Count)

- **정의**
	- 배송이이 완료된 주문의 총 개수

- **비즈니스적 의미**
	- 거래 발생 빈도를 나타내는 지표로, 매출 변화의 원인을 분해하기 위한 기준 지표

- **계산 대상(Granularity)**
	- 주문

- **계산 기준**
	- 주문 ID 기준 중복 제거 후 개수 산출

- **포함 조건**
	- 배송이 완료된 주문만 포함

- **시간 축 활용**
	- 주문 수 추이 분석
	- 매출 감소 시 주문 수 vs AOV 분리 해석


### 구매자 수 (Active Buyers)

- **정의**
	- 특정 기간 내 1회 이상 구매한 고유 고객 수

- **비즈니스적 의미**
	- 실제 구매 활동이 발생한 고객 규모를 나타내며, 고객 기반의 성장 또는 축소 여부를 판단하는 지표

- **계산 대상(Granularity)**
	- 고객

- **계산 기준**
	- 고객 고유 식별자 기준 중복 제거

- **포함 조건**
	- 배송이 완료된 주문 기준

- **시간 축 활용**
	- 연 / 월별 구매자 수 변화
	- 신규 / 이탈 신호 감지
	- 코호트 분석의 기준 KPI로 활용


### 재구매율(Repeat Purchase Rate)

- **정의**
	- 전체 구매 고객 중 2회 이상 구매한 고객의 비율

- **비즈니스적 의미**
	- 고객 유지 상태를 요약하는 핵심 지표로, 장기적인 비즈니스 안정성을 판단하는 데 활용

- **계산 대상(Granularity)**
	- 고객

- **계산 기준**
	- 2회 이상 구매 고객 수 / 전체 구매 고객 수

- **시간 축 활용**
	- 월별 재구매율 추이
	- Cohort 분석

## 2. 파생 KPI


```
파생 KPI는 핵심 KPI를 기반으로 계산되며,
비즈니스 성과의 원인을 보다 세분화하여 해석하기 위해 활용되는 지표입니다.

파생 KPI는 독립적인 집계 대상이라기보다는,
핵심 KPI의 변화를 설명하고 문제 구간을 식별하기 위한 보조 지표의 역할을 수행합니다.

- AOV (Average Order Value)
- 주문 취소율 (Order Cancellation Rate)
```

### AOV(Average Order Value)

- **정의**
	- 주문 1건당 평균 주문 금액

- **비즈니스적 의미**
	- 주문의 질적 수준을 나타내는 지표로, 할인·프로모션·상품 구성 전략의 영향을 분석하는 데 활용

- **계산 대상(Granularity)**
	- 기간 단위 집계 KPI

- **계산 기준**
	- 총 매출 / 주문 수

- **포함 조건**
	- 배송이 완료된 주문만 포함

- **시간 축 활용**
	- 매출 변화 원인 분석
	- 주문 수는 유지되나 매출이 변하는 경우 설명


### 주문 취소율(Order Cancellation Rate)

- **정의**
	- 전체 주문 중 취소 상태로 종료된 주문의 비율

- **비즈니스적 의미**
	- 주문 프로세스 상의 문제(결제 실패, 판매자/재고 이슈, 고객 이탈)를 종합적으로 파악하기 위한 지표

- **계산 대상(Granularity)**
	- 주문

- **계산 기준**
	- 취소 주문 수 / 전체 주문 수

- **포함 조건**
	- 주문 생성 기준 전체 주문

- **시간 축 활용**
	- 월별 취소율 추이
	- 카테고리/판매자/지역별 비교
